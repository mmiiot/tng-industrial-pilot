About

The Machine Simulator simulates an injection molding machine (IMM).

The Machine Simulator simulates machines:

  • with EM63 protocol support
    • using Samba for data exchange (default)
    • using local file share (../em63_share/) for data exchange (optional argument: --disableSamba)
    • using FTP for data exchange (currently not implemented)
  • with OPC UA protocol support for data exchange (optional: parameter --enableOPCUA)

How to setup the machine simulator

  • Go to Setup
  • Select Parameter
  • Modify Test parameter (optional)
  • Press Button Set Test Parameter Values
  • Select Job
  • Modify Job Parameter (optional)
  • Press Button Set Job Parameter
  • Select Machine State
  • Select Production
  • Go to Monitoring
  • Consider the machine states:

Machine States

Selectable machine states:

Idle/Setup

Production

Machine Error

Pause

Note:
  • Select Idle when a job was finished.
  • Select Machine Error to simulate a machine error.
  • Select Pause to pause the machine.
  • Select Production to start production or continue after pause or machine error.

 

Network

If the website configuration takes to much time for loading check your network configuration. If you use a static network configuration, check the gateway and DNS address. Problem occurs, if DNS and gateway address are set but not reachable.
